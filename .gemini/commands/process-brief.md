# /process-brief

執行以下流程處理 shared-memory/client/current/brief.md：

1. **COO 三問判斷** — 讀取 brief，回答三問，決定接案或婉拒
2. **派問題診斷師** — 分析 brief，產出問題定位草稿
3. **COO 審查診斷** — 確認問題定位是否清晰具體
4. **派思維訓練師 + 工具配置師** — 依診斷結果各自產出
5. **COO 整合** — 合併三份內容，確認邏輯一致
6. **輸出 deliverable** — 存到 shared-memory/client/current/deliverable.md
